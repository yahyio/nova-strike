# Nova Strike — Arcade Space Shooter

Canvas-based arcade shooter with inertia flight, particle explosions and a localStorage high score. No libraries, single file of game logic.

## Features

- Inertia-based flight controls
- Three enemy types: sine drifters, chasers, rotating tanks
- Particle explosions and screen shake
- Wave scaling difficulty
- LocalStorage high score

## Tech Stack

HTML · Canvas 2D · Vanilla JavaScript

## Run

Open `index.html` directly in any browser.
